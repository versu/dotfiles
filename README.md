# Requirement

Ubuntu-20.04

# How to use

## 1. install wsl2 + ubuntu

```powershell
wsl --install -d Ubuntu-20.04
```

after install. pc restart.
set username and password.

bash <(curl -LSs https://raw.githubusercontent.com/versu/dotfiles/main/install.sh)
